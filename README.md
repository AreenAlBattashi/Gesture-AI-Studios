# Gesture AI Studios

Gesture AI Studios is an AI-powered interactive smart mirror platform that uses computer vision to recognize gestures and trigger customized actions.

The administrator can create a project, select gestures, and decide what happens when users perform those gestures, such as displaying text, images, videos, or effects.

## Target Users
- Teachers
- Event coordinators
- Businesses
- Exhibition organizers
- General users

## Tech Stack
- Python
- OpenCV
- MediaPipe
- LLM/API integration
- Streamlit or simple UI
