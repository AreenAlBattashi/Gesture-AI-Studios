# 📖 Overview

Gesture AI Studios is an AI-powered Smart Mirror platform that enables users to create customizable, touchless interactive experiences using real-time hand gesture recognition.

Instead of writing code, users can configure gesture actions through an intuitive interface, making the platform suitable for exhibitions, museums, classrooms, retail displays, digital signage, and interactive kiosks.

The application combines real-time computer vision with an easy-to-use dashboard, allowing users to personalize gesture behavior, manage multiple projects, and launch an interactive Smart Mirror experience.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/AreenAlbattashi/Gesture-AI-Studios.git
```

Move into the project

```bash
cd Gesture-AI-Studios
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run Src/app.py
```

# 🧠 How It Works

1. Create an account or log in.
2. Customize gesture actions.
3. Save your project configuration.
4. Launch the Smart Mirror.
5. Perform gestures in front of the camera.
6. Watch the configured actions execute in real time.
